Data Mining

คือกระบวนการค้นพบความรู้และรูปแบบที่ซ่อนอยู่ภายในข้อมูล

เป็นการดึงเอาข้อมูลหรือแพทเทิร์นที่มีประโยชน์ออกมาจากข้อมูลจำนวนมหาศาล

Data Mining ในบริบทของ Business Intelligence

ลำดับขั้นทั่วไป:
Data Source → การเตรียมข้อมูล/การผสานข้อมูล, Data Warehouse → การสำรวจข้อมูล → การทำเหมืองข้อมูล → การนำเสนอผลลัพธ์ → การตัดสินใจ

ประเภทของข้อมูลที่สามารถนำมาทำ Data Mining ได้

  1.ข้อมูลในฐานข้อมูล (Database)
  
  2.ข้อมูลในคลังข้อมูล (Data Warehouse)
  
  3.ข้อมูลธุรกรรม (Transactional data)
  
  4.ข้อมูลแบบสตรีม (Stream data)
  
  5.ข้อมูลเชิงพื้นที่และเวลา (Spatiotemporal)
  
  6.ข้อมูลอนุกรมเวลา (Time-Series)
  
  7.ข้อมูลลำดับเหตุการณ์ (Sequence data)
  
  8.ข้อความและข้อมูลจากเว็บ (Text & Web)
  
  9.ข้อมูลมัลติมีเดีย (Multimedia)
  
  10.กราฟ เครือข่ายสังคม และเครือข่ายข้อมูล (Graphs & Social Networks)
| ฟังก์ชัน                                                           | คำอธิบาย                                                                                             |
| ------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| **Generalization**                                                 | เป็นการสรุปลักษณะสำคัญของข้อมูล เพื่อนำไปสร้างกฎหรือข้อกำหนดสำหรับกลุ่มข้อมูลเป้าหมาย                |
| **Pattern Discovery**                                              | ใช้ค้นหาความสัมพันธ์ระหว่างรายการที่มักปรากฏร่วมกัน เช่น การวิเคราะห์ตะกร้าสินค้า                    |
| **Classification**                                                 | แบ่งข้อมูลออกเป็นกลุ่มตามประเภทหรือคุณลักษณะที่กำหนดไว้ล่วงหน้า                                      |
| **Cluster Analysis**                                               | จัดกลุ่มข้อมูลที่มีความคล้ายคลึงกัน โดยไม่ทราบป้ายกำกับล่วงหน้า                                      |
| **Outlier Analysis**                                               | ตรวจจับค่าผิดปกติ เพื่อประเมินคุณภาพข้อมูล และหลีกเลี่ยงการใช้ข้อมูลที่มีค่าผิดปกติสูงในการวิเคราะห์ |
| **Time & Ordering Analysis**: Sequential Pattern, Trend, Evolution | วิเคราะห์ทิศทาง แนวโน้ม และลำดับเหตุการณ์ เช่น การคาดการณ์ด้วยโมเดลถดถอยจากข้อมูลในอดีต              |
| **Structure & Network Analysis**                                   | การทำเหมืองข้อมูลบนโครงสร้างข้อมูล เช่น กราฟ เครือข่ายข้อมูล และข้อมูลเว็บ                           |
Data Mining: การผสานหลายศาสตร์

เป็นการรวมแนวคิดและเทคนิคจากสาขาต่างๆ เข้าด้วยกัน เพื่อดึงข้อมูลเชิงลึกและค้นหารูปแบบที่มีคุณค่าออกมาจากฐานข้อมูลขนาดใหญ่
